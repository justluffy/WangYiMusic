## 高仿网易云音乐

* 数据源自[网易云音乐 API](https://github.com/Binaryify/NeteaseCloudMusicApi)
* swift 5编写。
* StreamingKit实现音频。
* ijkplayer实现卡片视频。
* 主要实现了主页，歌单，播放器
* 我在B站上传了播放器的清晰版：[链接](https://www.bilibili.com/video/BV1A54y1W7TT)
